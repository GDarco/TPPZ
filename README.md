# Smart-Energy Monitoring System

## Опис проєкту
Програмне забезпечення для збору та аналізу даних з розумних лічильників електроенергії. 
Система дозволяє відстежувати споживання в реальному часі та формувати звіти.

## Структура репозиторію
- `src/main/java` — основна логіка системи (Java).
- `docs/` — технічна документація та схеми архітектури.
- `configs/` — шаблони конфігураційних файлів.

## Технологічний стек
- Java 17
- Spring Framework
- PostgreSQL
